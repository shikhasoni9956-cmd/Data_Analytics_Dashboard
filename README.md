# Student Performance Analytics Dashboard

## Project Description
The Student Performance Analytics Dashboard is a web-based application developed using Python and Streamlit. It helps analyze student performance using interactive charts and visualizations.

## Features
- Student Performance Analysis
- Interactive Dashboard
- Charts and Graphs
- Data Filtering
- CSV File Upload
- Performance Reports

## Technologies Used
- Python
- Streamlit
- Pandas
- Plotly

## Project Structure
- app.py
- requirements.txt
- pages/

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
